## 📊 Data Requirements & Formats

### **Twitter/X Analytics CSV:**
**Required columns:**
- `Date` - Post date (YYYY-MM-DD format)
- `Impressions` - View count (numeric)

**Optional columns:**
- `Clicks` - Click count (if available in CSV)
- `Engagements` - Total engagement metrics
- `Profile_Visits` - Profile click metrics

### **Click Data Input:**
- **Manual Entry:** Enter clicks for each date via the interface
- **CSV Upload:** Separate CSV with Date and Clicks columns  
- **Date Matching:** Auto-matches dates between impression and click data
- **Data Validation:** Built-in validation and error handling

### **Supported Formats:**
- 📄 CSV files (comma-separated)
- 📅 Date formats: YYYY-MM-DD, MM/DD/YYYY, DD/MM/YYYY
- 🔢 Numeric formats: Raw numbers, formatted numbers (1,000)

## 🐛 Troubleshooting

### **Common Issues:**

**❌ CSV Upload Not Working**
- ✅ Ensure CSV has required columns: Date, Impressions
- ✅ Check date format is consistent (YYYY-MM-DD preferred)
- ✅ Remove any special characters from data
- ✅ Try refreshing the page and re-uploading

**❌ Charts Not Displaying**
- ✅ Check browser console for JavaScript errors
- ✅ Ensure browser supports modern JavaScript (ES6+)
- ✅ Try disabling browser extensions temporarily
- ✅ Use Chrome/Firefox for best compatibility

**❌ Export Not Working**
- ✅ Allow pop-ups for the domain
- ✅ Check browser download settings
- ✅ Try right-click > "Save As" on generated content
- ✅ For PDF: Use Chrome's print functionality

**❌ Fonts Not Loading**
- ✅ Fonts are embedded - should work offline
- ✅ Check network connection for Google Fonts fallbacks
- ✅ Clear browser cache and reload
